🐛 3 BUGS CRÍTICOS CORRIGIDOS
────────────────────────────────────────────────────────────────────────

BUG 1: BROADCAST Format Redundante
├─ Ficheiro: client_linux.c, linha 1045
├─ Problema: "BROADCAST #geral msg" (canal enviado 2x)
├─ Solução: "BROADCAST msg" (sem canal redundante)
├─ Validação: ✅ BCAST_SENT recebido
└─ Teste: (echo "BROADCAST test") | nc 127.0.0.1 10000

BUG 2: LOGOUT Handler Missing
├─ Ficheiro: server_linux.c, linhas 1328-1329
├─ Problema: LOGOUT retornava CMD_INVALID
├─ Solução: Adicionar strcmp(buffer, "LOGOUT") handler
├─ Validação: ✅ LOGOUT_OK recebido
└─ Teste: (echo "LOGOUT") | nc 127.0.0.1 10000

BUG 3: Client Blocks on fgets()
├─ Ficheiro: client_linux.c, linha 1010
├─ Problema: Chat bloqueava no teclado (sem duplex)
├─ Solução: select() monitorar stdin + socket simultaneamente
├─ Validação: ✅ Broadcasts chegam durante digitação
└─ Teste: Conexão persistente com multi-mensagens
